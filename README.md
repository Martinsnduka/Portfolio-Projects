This project explores an E-commerce dataset using SQL to uncover insights on customer behaviour , order trends, and product performance.
**Dataset** : E-commerce transactions (customers, products,fact_sales).
**Tool used**: SQL Server Management studio.
**Key Questions Answered** :
**(1)** Top 5 best performing products that generated the highest revenue?(Found in E-commerce Data_Exploration.sql).
**(2)** Worst Performing Products in terms of sales?(Found in E-commerce Data_Exploration.sql).
**(3)** Top customers by order value?(Found in E-commerce Data_Exploration.sql).
**(4)** Analyze the sales performance over time?(found in SQL_Advanced_EDA.sql).
**(5)** Analyze the yearly performance of products by comparing their sales to both the average sales performance of the product and the previous year's sales?(found in SQL_Advanced_EDA.sql).
**(6)** Identify the top performing products categories as percentage of total sales? (found in SQL_Advanced_EDA.sql).
**(7)** Segment products into cost ranges and count how many products fall into each segments?(found in SQL_Advanced_EDA.sql).
